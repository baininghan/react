技术支持

React 由 Facebook 的 product infrastructure 和 Instagram 的 user interface engineering 团队提供支持(full-time). 他们会随时关注各种新问题.

Stack Overflow #

很多程序员通过 Stack Overflow 来提出和回答问题. 你可以先搜索 有人问过的 reactjs 问题, 或者 新开一个问题 !

Google Groups mailing list #

The reactjs Google Group 也是一个提问或搜索答案的好地方.

IRC #

还有很多开发者闲着没事干就会逛逛 Freenode.net 的 IRC network: #reactjs on freenode.

Twitter #

你可以在 Twitter 上关注 React 的最新信息: #reactjs on Twitter.
