React
一个创建用户界面(UI)的 JavaScript 库
快速入门 下载 React v0.12.2

Just the UI

很多人将 React 用作 MVC 中的 V(View,视图,展示界面). 因为使用 React 不要求掌握其他的特定技能, 所以集成到一个现有项目上是很容易的.

Virtual DOM

为了暴高的性能(ultra-high performance), React 使用一个 virtual DOM 的不同实现. 也可以通过 Node.js 在服务端进行渲染 — 不再需要笨重的 browser DOM.

Data flow

React 实现了单向(one-way)的数据流反应, 这能减少样板文件,比起传统的数据绑定更容易推导和分析(reason about).

一个持有状态(Stateful)的组件

除了接受输入数据 (通过 this.props 获取), 一个组件可以维护内部的状态数据 (通过 this.state 得到). 当一个组件的状态数据(state data)发生改变, 通过再次调用 render() 方法,原来渲染生成的标签(rendered markup)将被更新.
